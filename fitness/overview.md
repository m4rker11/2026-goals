# Fitness: 200 Minutes Weekly

## Overview

| Current | Target | Timeline |
|---------|--------|----------|
| ~50 min/week | 200 min/week | 8 weeks |

---

## The System

**Split:** Upper / Mid / Lower / Cardio (mix and match)

| Type | Focus | Examples |
|------|-------|----------|
| Upper | Push + Pull | Pushups, pullups, dips, rows, shoulders |
| Mid | Core + Stability | Dead bugs, planks, bird dogs, PT exercises |
| Lower | Legs | Squats, lunges, RDLs, calves |
| Cardio | Heart + Lungs | Zone 2, HIIT, swimming, jogging, OT |

## Success Metrics

**Weekly:** Hit target minutes (within 80% counts as success)

**Key Numbers to Track:**
- Pushups: max reps in set 1
- Pullups: max reps in set 1
- Session length: building toward 60 min
